# post-to-tinjs
ServiceTIN สำหรับตรวจสอบเลขประจำตัวผู้เสียภาษีอากร (13 หลัก) ว่าถูกต้องและมีอยู่จริงในฐานข้อมูลหรือไม่

## Installation
```
npm install tin-id
```
## Usage
```
import { CheckTaxID } from "tin-id"

let tax = "" //Tax Identification Number
CheckTaxID(tax)

```
